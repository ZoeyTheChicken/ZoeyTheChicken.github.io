Hey guys! The MacBook Air mid 2013 has decreased in price again by about $20!


They can now be found for about $80 with some configured with an upgraded 256GB SSD.


The mid 2013 MacBook Air still holds up amazingly 11 years later running macOS Big Sur, you can run almost any modern application you need.


Keep in mind, macOS Big Sur is a few years old now, meaning a few Apple apps like Xcode, Safari, etc. Don't support macOS Big Sur on the latest version. Although Safari 16.6.1 works just as good as Safari 17.


The newest Xcode version for Big Sur is 13.2.1, so if you are a developer, consider a macOS Ventura compatible device.


For our dev friends, there is a tool called Open Core Legacy Patcher which is available on GitHub, it allows the installation of macOS Ventura, even Sonoma on old Macs such as the 2013 MacBook Air.


If you want to run macOS Ventura, or Sonoma using OCLP on your mid 2013 MacBook Air, I would suggest purchasing the upgraded version of it with 8GB RAM and an i7 instead, you can pick up the upgraded i7 option for about $30 more.


If you need a lot of storage, this mac model is probably the perfect option, it can be found with up to 512GB of Storage.


The best version of this special laptop in MY opinion is the 13 inch 512GB SSD, i7 configuration with 8GB of RAM. If you find that at a reasonable price range and use OCLP, expect a computer that can hold up good for years to come.


Even though this device is turning 11 this year, it still OFFICIALLY runs the 3rd newest macOS build and can run the newest one with OCLP! Quite a great machine.


If you are looking to save money and don't need the best specs, the base model 11" Core i5 with 4GB RAM and a 128GB SSD should meet your needs.

Machines like this are amazing, capable machines but if you want better performance from the i7-4650U configuration, I would recommend the 2015 MacBook Air, it comes with up to an i7-5650U and has native macOS Monterey support so it is less likely you need to install OCLP.


Macs from Air 2011-2015 come with UP TO an i7 CPU, Macs from Air 2016-2017 Air come with an i7 CPU by default. Macs from Air 2020_m1-2022 come with M1/M2 CPU's


Macs from Air 2010-2011 came with 128/256GB options, 2012 Air was first to have 512GB option. Macs from Air 2018 began coming with 1TB of storage.


Again, 2012-2016 MacBook Air's are an insane deal! I'd recommend picking one up!
